Berlin Uhr Clock Implementation in C#
This is a console application written in C# using Factory Design Pattern and Dependcy Injection.
Github Repo Link: https://github.com/shailesh91/Berlin-Clock
There are two methods to run the application:
1. Clone the Github Repo and import it to a C# compatible IDE and simply run the Application.
2. Clone the Github Repo and navigate to Berlin-Clock/Berlin-Clock/obj/x86/Debug/ and find Berlin-Clock.exe. Download this and simply run it to start the console application.
Testing:
Although a better approach would be to write unit test cases which makes certain assertions. 
To test the application, enter different formats and Input Strings and check against an expected output on the console. 
If their is some error in format or supported input or output formates, exceptions is thrown and program terminates.
